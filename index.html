Get a free key at openrouteservice.org → Dashboard → API Keys.
Free tier: 2,000 direction requests/day. With a valid key the tool fetches real road distances (HGV profile) for every route — making dispatch times significantly more accurate. Without a key it uses straight-line × 1.35.

Per route calls ORS once per generated route to get total km + drive time.
Per stop calls the ORS Distance Matrix API to get real inter-stop distances before clustering — best results but uses more API quota.

How it works
Your key is only sent directly to api.openrouteservice.org from your browser — it is never stored on a server. Route results are cached per session to avoid repeat calls.
